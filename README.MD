# Fasty-gql [WIP]

| A Fastify 🚀 + GraphQL Yoga 🧘‍♀️ Server

## Features

- [x] Typescript Codegen
- [x] Modules using graphql-modules
- [ ] GraphQL Middlewares
- [ ] Logs
- [ ] Graphql Subscription

## Developemt

### Start Server for development

```bash
 pnpm dev
```

### Generate Types

```bash
pnpm codegen
```

## Production

### Build app for production

```bash
pnpm build
```

### Start server

```bash
pnpm start
```
